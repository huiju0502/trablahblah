## 📝 개요
- 여행 커뮤니티 게시판
- 2023년 5월 2일 ~ 2023년 5월 15일
- 개인 프로젝트

## 💻 기능설명
- 로그인 / 로그아웃
- 회원가입 / 탈퇴
- 마이페이지
  - 비밀번호 변경 가능
  - 새로운 비밉번호 일치여부 유효성 검사
- 지역 카테고리
  - admin(관리자) 계정만 카테고리 관리 버튼 활성화  
  - 카테고리 별로 게시글 출력
  - 카테고리 추가시 중복검사
  - 카태고리내에 게시글이 하나라도 있으면 카테고리 삭제 불가
- 게시글 작성
  - 로그인 시에만 NEW버튼 활성화되게 session 분기
  - 작성자만 수정 / 삭제 버튼 활성화되게 session 분기
- 댓글 작성
  -  게시글에 댓글 작성 기능
  -  댓글 작성자만 수정 / 삭제 버튼 활성화되게 session 분기
